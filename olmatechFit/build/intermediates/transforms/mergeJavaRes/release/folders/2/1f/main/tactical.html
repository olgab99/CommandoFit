WORKOUT SCREEN FOR TACTICAL WORKOUT

On this screen you see a list of exercises you should perform in a sequence without rest between them.

LAPS box contains number of times you completed all exercises in the list.There are two types of exercises in the list – cardio and weight.

  • For Cardio exercise tap START button to start, cursor will advance to the next exercise when time elapses.
  • For Weight exercise – tap DONE button to mark exercise as completed.

Tap DONE WORKOUT button when you finish workout. Exercise list and number of laps completed will be saved to the workout log.
