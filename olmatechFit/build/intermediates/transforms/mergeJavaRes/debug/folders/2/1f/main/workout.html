WORKOUT SCREEN

For complete program, and Endurance and Strength modules workout screen displays current exercise – cardio or weight.

For Tactical module workout screen displays a list of the exercises.

Workout Screen for Complete Program, Strength / Endurance and Mass / Strength Modules:

You can do exercises in any order, but pre-set order is highly recommended.

Cardio exercise view displays recommended time interval.

  • If you wish to change recommended time, use [+] and [-] buttons.
  • Tap START button when ready to start.
  • Tap DONE when finish.
  • SAVE button will become visible in the bottom right corner of the screen.
  • If you wish you can input distance and select cardio equipment used. Tap SAVE button to save time, distance and equipment used to the workout log.

Weight exercise view displays exercise image with starting and ending exercise positions, number of recommended repetitions (highlighted range on the "REPS" wheel) and sets. Tap blue arrow button to view exercise description.

Recommended weight calculator

When you open program for the first time:

Weight number input wheel will show "BW" (body weight); Reps wheel will be in range 12 – 18 for endurance training, and 6 – 12 for mass workout.

Choose the weight with which you think you can do 2 – 15 repetitions, do set to failure, enter the weight and number of repetitions with wheels, press

CALCULATE/SAVE button on the right.

Program will display recommended ranges of weight and repetitions for this exercise. Calculations depends on your fitness level that you entered during initial interview, so make sure to update it as you progress.

When you open exercise subsequent time:

Number input wheels will display recommended weight and number of repetitions. If your actual used weight and number of repetitions is different from recommended, enter them with corresponding wheels and tap CALCULATE/SAVE button.

NOTE: If you did less or more repetitions than recommended, "weight" wheel will be updated to display new recommended for you weight value.

Saved log entries will appear in the list next to CALCULATE/SAVE button, tap "pencil" icon to view log details.
