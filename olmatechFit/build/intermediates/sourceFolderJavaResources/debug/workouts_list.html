WORKOUTS LIST SCREEN

This screen displays workouts information on the left and exercise list on the right. The screen looks slightly different for Complete Program and Modules workouts.

Workout List screen for Complete Program:

The screen displays program schedule on the left and list of the exercises for the current workout on the right.

Different types of workouts marked with following symbols:

- Endurance workout.

- Strength workout.

- Tactical workout.

- Cardio workout.

The completed workouts are marked with shading and check-mark symbol. Current block of 7 days is highlighted in light-grey and current day has an orange arrow pointing to it.

Long-tap on any day with workout to open sub-menu:

  • View – tap to view workout exercises list.
  • Make Day Current – "roll back". Tap to set this day as your current day if you decided to repeat part of the program. All workout logs for the following days will be deleted.

    • Exercises list shows exercises for the current workout. Tap exercise in the list to go to "Workout" screen and start exercising. Long-tap exercise in the list to set its status to "Complete", or "Reset" it back to "not complete state".

Workout List Screen for Modules:

The screen contains list of workouts for selected module. Selected workout highlighted in light-grey and has an orange arrow pointing to it. Tap "Start" to go to "Workout" screen and start exercising, or "Select" to select workout.
