MENU button gives access to following:

  • Interview screen
  • Settings screen
  • Program Calendar and Workouts Logs screen
  • Reset Current Workout Log (available from Workout screen) - you can delete all workout logs for current exercise program and set it to start (Block 1 Day 1)
  • Reset Exercise Program
  • Remove Recommended Weight Values – you can remove all saved recommended weight values from all exercises
  • Remove Current Recommended Weight Value – you can remove saved recommended weight value for current exercise (available only from Workout screen)
  • Music - tap to open Media Player application.
  • Finish Workout and Go Back to Home Screen – accessible from Complete Program Workout Screen, you can set current workout complete and close Workout screen
  • Post to Facebook - you can post status update or picture to your Facebook page
