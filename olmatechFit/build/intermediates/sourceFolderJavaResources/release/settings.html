SETTINGS SCREEN

On this screen you can set / modify following preferences:

  • Rest interval between sets - Following completion of each set, you can use timer to time the rest interval between sets – choose 30, 45, or 60 seconds interval
  • Play sound on time up – check it if you wish sound notification when rest and cardio time is up
  • Start timer on log save – check it if you wish to starts timer automatically when you tap Calculate/Save button
  • Advance to the next exercise when recommended number of sets completed – check it if you wish automatically advance to the next exercise after you completed recommended number of sets and saved them to workout log
  • Choose one of the Units radio buttons to use kilograms or pounds, miles or kilometres to measure weights and distances
