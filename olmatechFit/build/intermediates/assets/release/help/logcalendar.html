PROGRAM CALENDAR AND WORKOUTS LOGS SCREEN

This screen displays monthly calendar showing your saved workout logs (the days with saved workout logs marked with "document" icon). On first screen opening, logs for a current month for Complete Program mode and recommended program schedule are displayed. Use FILTER drop-down list to select type of the workouts.

Toggle "Show program schedule" check-box to display / hide proposed schedule for forthcoming training and rest days.

Swipe calendar to show next or previous month or tap Change Date to select month and year. Tap on day with "document" icon to view Workout Log for his day.

WORKOUTS LOG VIEW SCREEN

This screen displays workout log. You can email log from this screen (it is recommended to have gmail installed on your phone for this functionality to work properly).
